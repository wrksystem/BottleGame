# BottleGame

Um pequeno jogo que gira a garrafa de forma aleatória.

# Requisitos de Sistema

° Android 7.0 ou superior

# Conclusão
Um simples joguinho para praticar manipulação de eixo X e Y dentro do Android com Java..
